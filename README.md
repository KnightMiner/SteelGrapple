# Steel Grapple
This mod adds adds several tweaks to the Into the Breach Steel Judoka squad, and additionally adds a new mech option to that squad. For more information, see the [forum thread](https://subsetgames.com/forum/viewtopic.php?f=25&t=35584) on the Subset Games forum.

## Installation
1. Install [kartoFlane's modloader](https://github.com/kartoFlane/ITB-ModLoader)
2. Download the latest version from the [releases](https://github.com/KnightMiner/SteelGrapple/releases), or click "Clone or download" then "Download ZIP" to get the latest builds.
2. Unpack the archive to your mods directory (<steam>\steamapps\common\Into the Breach\mods).
